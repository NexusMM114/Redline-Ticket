# 🔥 RedLine Ticket Bot (Pro Version)

A powerful Discord ticket system built with:
- Node.js
- Discord.js v14
- MongoDB Atlas
- Mongoose
- HTML Transcript System
- Review & Rating System

---

## 🚀 Features

✅ Admin-only `/panel` command  
✅ Ticket dropdown system  
✅ Private ticket channels  
✅ Duplicate ticket prevention  
✅ Claim system (staff only)  
✅ Only claimed staff can close  
✅ HTML transcript generation  
✅ Review system (1–5 rating via DM)  
✅ MongoDB ticket storage  
✅ Automatic ticket deletion
